## ğŸ› TROUBLESHOOTING

### Call not triggering?
1. Check GHL workflow is published
2. Verify webhook URL is correct
3. Check server logs for errors

### Tool not working?
1. Verify tool is in assistant config
2. Check webhook URL matches deployed server
3. Test endpoint directly with Postman

### No SMS sent?
1. Check Twilio credentials
2. Verify phone number format (E.164)
3. Check GHL automation triggers

### Payment not updating GHL?
1. Verify Stripe webhook is configured
2. Check `contactId` in Stripe metadata
3. Review server logs for Stripe events
